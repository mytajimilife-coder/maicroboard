Quick Start

Installation

  1. Download or clone the repository
  2. Upload files to your web server
  3. Navigate to http://your-domain.com/install.php
  4. Follow the installation wizard
  5. Configure database settings and admin account
  6. Complete installation and start using MicroBoard!

Requirements

  • PHP 7.4 or higher
  • MySQL 5.7 or higher / MariaDB 10.2 or higher
  • Apache or Nginx web server
  • PDO PHP Extension
  • cURL PHP Extension (for OAuth)

OAuth Configuration

After installation, configure OAuth providers in the admin panel:

  1. Login as admin
  2. Go to Admin Panel ‚Üí OAuth Settings
  3. Enter Client ID and Client Secret for each provider
  4. Enable the providers you want to use
  5. Social login buttons will automatically appear on login/register pages

üìñ Documentation: See OAUTH_SETUP.md for detailed OAuth setup instructions.
